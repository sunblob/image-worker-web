# shadcn/ui Redesign — Design Spec

**Date:** 2026-03-26
**Status:** Approved

---

## Overview

Redesign the Image Worker web app with a modern dark UI using shadcn/ui components. The app has two tools — **Compress** and **Edit** — accessed via a collapsible sidebar. The main content area centers around a large, inviting file upload zone.

---

## Theme

- **Mode:** Dark only (no light/dark toggle needed)
- **Background:** `#080808` (body), `#111` (sidebar, cards)
- **Border color:** `#1f1f1f`
- **Accent:** Indigo — `#6366f1` (primary), `#818cf8` (hover/muted), `#a5b4fc` (active text)
- **Success:** `#4ade80` (savings percentage in compress queue)
- **shadcn theme:** Install with `dark` class on `<html>`, customise CSS variables to match the above palette

---

## Sidebar

**Behaviour:** Collapsible. State stored in `useState`. Defaults to expanded.

**Expanded (220px wide):**
- Top row: PanelLeft icon button (toggles collapse) + app logo icon + "Image Worker" wordmark
- Nav items: 32×32 icon box + label text, indigo active highlight, left active bar
- Smooth CSS transition on width (`220px → 52px`)

**Collapsed (52px wide):**
- Top row: PanelLeft icon button only (logo/wordmark hidden via `opacity: 0`)
- Nav items: icon only, centered in the rail; active state = `bg-indigo-500/20` on the icon box; no stretched row background
- Active bar hidden when collapsed
- Tooltips on nav icons showing the label (shadcn `Tooltip` component)

**Components used:** `Button` (variant ghost, for toggle + nav items), `Tooltip`, `TooltipProvider`

## Drop Zone

Used in both Compress (multi-file) and Edit (single file) pages.

**Appearance:**
- Full-width up to `max-w-2xl`, height `h-56`
- `border-2 border-dashed border-indigo-500` with `rounded-2xl`
- Radial gradient background glow: `radial-gradient(ellipse at 50% 60%, rgba(99,102,241,0.08) 0%, transparent 70%)`
- Centered content: upload arrow icon in a soft indigo circle, bold title, muted subtitle

**Subtitle text:** `or click to browse · paste from clipboard`

**States:**
- Default: indigo dashed border
- Drag-over: `border-indigo-400`, brighter glow, subtle scale transform
- Disabled: `opacity-50 pointer-events-none`

**Clipboard paste:**
- Global `paste` event listener attached in a `useEffect` on the page (not inside DropZone — keeps the listener at page scope)
- Extracts `image/*` items from `event.clipboardData.items`
- Calls the same `onFiles` handler as drag-and-drop
- Brief visual flash on the drop zone (`ring-2 ring-indigo-400`) for 600ms to confirm paste was received

## Compress Page

**Layout:** Single column, max-width `2xl`, centered.

**Format/Quality controls:** Rendered as pill chips above the drop zone using shadcn `Select` and `Slider`. Labels are small uppercase. The quality slider shows the numeric value inline.

**Job queue:**
- Each row: file icon, filename (truncated), size before → after, savings % in green, Download button
- Processing state: indigo pulsing "Processing…" text
- Error/expired states: muted red or gray text
- "Download all (N)" button (indigo filled) top-right of queue, only visible when ≥1 job is done

---

## Edit Page

**Layout:** Two-column when an image is loaded (preview left, controls right). Drop zone centered when no image loaded.

**Controls panel (right, `w-64`):** Sections — Output (format, quality), Dimensions (width, height, fit), Filters (blur slider, sharpen/grayscale switches, rotate select, flip select). Apply + Reset buttons at the bottom. Uses shadcn `Select`, `Slider`, `Switch`, `Button`.

**Image preview (left):** Fills remaining space, image displayed with `object-contain`, filename + size metadata below.

## Clipboard Paste — Implementation Detail

In `CompressPage` and `EditPage`, attach a `paste` listener in `useEffect`:

```ts
useEffect(() => {
  function onPaste(e: ClipboardEvent) {
    const items = Array.from(e.clipboardData?.items ?? [])
    const files = items
      .filter(i => i.kind === 'file' && i.type.startsWith('image/'))
      .map(i => i.getAsFile())
      .filter(Boolean) as File[]
    if (files.length) handleFiles(files)
  }
  document.addEventListener('paste', onPaste)
  return () => document.removeEventListener('paste', onPaste)
}, [handleFiles])
```

The DropZone receives a `flashing` prop (boolean) that applies the ring style for 600ms after a paste.
